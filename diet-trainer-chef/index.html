ヘルシー鶏胸肉のねぎ塩豆腐焼き

15分 P: 35g / F: 8g / C: 12g 260 kcal

材料

・鶏胸肉: 150g ・木綿豆腐: 100g ・ネギ: 1/2本 ・塩コショウ、ごま油: 少々

手順

  1. 鶏胸肉をそぎ切りにし、塩コショウをする。
  2. 豆腐は水切りして一口大に切る。
  3. フライパンにごま油を熱し、鶏肉と豆腐を焼く。
  4. みじん切りにしたネギと塩で調味する。

👨‍🍳 シェフの一言

「鶏胸肉は火を通しすぎないのが、しっとり美味しく仕上げる秘訣です。豆腐でかさ増しすることで満足感が高まりますよ！」

AI機能の有効化 (Gemini API)

食事写真の解析やレシピの生成に Gemini API を使用します。お手持ちの Gemini API キーを入力してください。
※キーは暗号化されず、ブラウザの LocalStorage にのみ直接保存されます（外部送信されません）。

💡 デモモードについて

APIキーを設定しない場合は、デモモードで動作します。
・食事解析：ダミーのハンバーグ定食をシミュレート
・レシピ提案：固定の鶏胸肉レシピを出力
・チャット：トレーナーとシェフからのテンプレート返答
本物のAIパーソナルトレーニングを体験するには、Gemini APIキーの取得をおすすめします。
