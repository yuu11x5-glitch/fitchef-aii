# FitChef AI (フィットシェフ AI)

**FitChef AI** は、専属の「パーソナルトレーナー」と「専属シェフ」が合わさった、あなた専用のAIダイエットサポートアプリです。ブラウザで開くだけで即座に動作する、スタイリッシュなシングルページアプリケーション (SPA) です。

---

## 🌟 主な機能

1. **食事解析 AI (Diet AI)**
   - 食べたものの写真をアップロードまたはドラッグ＆ドロップするだけで、AI（Gemini 2.5 Flash）が料理名、推定カロリー、PFCバランス（タンパク質、脂質、炭水化物）を自動算出します。
   - トレーナーのレオから「運動・生活のアドバイス」、シェフのレイから「ヘルシー調理のアプローチ」がそれぞれ個別に提示されます。

2. **筋トレ管理 (Workout Tracker)**
   - 筋トレのメニューや重量、回数、セット数、消費カロリーを簡単登録。
   - トレーナーがあなたの目標に合わせた「今日のおすすめトレーニング」をアドバイスします。

3. **専属シェフの自炊レシピ提案 (Chef's Kitchen)**
   - 「冷蔵庫にある食材」「調理時間」「レシピの方向性（高タンパク、低糖質など）」を入力すると、AIシェフが現在の摂取目標に合わせたオリジナルヘルシーレシピを瞬時に考案します。

4. **トレーナー＆シェフのAIチャット (Trainer Chat)**
   - 運動、食事、ダイエットの悩みについてAIコーチと直接チャット相談できます。
   - 「音声で聞く」を有効にすると、ブラウザの音声合成機能（Web Speech API）により、アドバイスを音声で読み上げてくれます。

5. **目標・PFC・データの管理 (Dashboard & Settings)**
   - ダッシュボードの進捗ゲージ、および Chart.js を用いた美しいPFCバランスのドーナツグラフで、今日の摂取状況を一目で把握。
   - すべてのデータはブラウザの `LocalStorage` に安全に保管されます（外部サーバーへの送信は一切ありません）。

## 🚀 起動方法

### 方法 1: ダブルクリックで起動（最も簡単）
1. プロジェクトフォルダ内の `index.html` ファイルをダブルクリックして、お好きなブラウザ（Chrome, Edge, Safari等）で開きます。
2. すぐにアプリが起動し、すべてのデザインと機能（デモモード）をお試しいただけます。

### 方法 2: ローカルサーバーで起動
もしローカルサーバーから起動したい場合は、VS Codeの「Live Server」拡張機能を使用するか、ターミナルで以下を実行します：
```bash
# Pythonがインストールされている場合
python -m http-server 8000

# Node.js (npx) がインストールされている場合
npx serve
```
その後、ブラウザで `http://localhost:8000` または `http://localhost:3000` にアクセスしてください。

## 🔑 AI機能を本物にする（Gemini APIキーの設定）

本アプリは、APIキーを設定しない場合は**デモモード**（あらかじめ用意された疑似的な解析結果やテンプレート回答）で動作します。

本物のAI（画像解析、レシピ提案、フリートーク）を利用するには、以下の手順で Gemini API キーを設定してください。

1. **APIキーの取得**:
   - [Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインして、無料の Gemini API キーを発行します。
2. **アプリでの設定**:
   - アプリの左側メニュー（スマホの場合は下部ナビ）から「**設定**」を開きます。
   - 右側の「**AI機能の有効化**」エリアに、取得した API キー（`AIZAaSy...` で始まる文字列）を貼り付けます。
   - 「**キーを保存**」ボタンをクリックします。
3. これで、本物の食事写真の解析や、入力した食材に合わせた無限のオリジナルレシピ考案、リアルタイムなコーチとの対話が可能になります！
